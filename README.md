## вңЁ Tentang Project

**Wanderlust Indonesia** adalah web app quiz interaktif yang membantu kamu menemukan destinasi liburan paling cocok di seluruh Indonesia. Jawab 20 pertanyaan seputar gaya hidup, preferensi cuaca, tipe aktivitas, dan kepribadianmu lalu dapatkan rekomendasi personal yang dipersonalisasi khusus untukmu.

> *"Jangan tanya kenapa harus ke Indonesia - tanya kenapa belum."*

---

## рҹҺҜ Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| рҹҺІ **20 Pertanyaan Cerdas** | Pertanyaan terstruktur dalam 8 kategori: Gaya Hidup, Petualangan, Kuliner, Cuaca, dan lainnya |
| рҹ—әпёҸ **50+ Destinasi Indonesia** | Database lengkap dari Sabang sampai Merauke, dikategorisasi dan ditagging |
| рҹ§  **Sistem Scoring Pintar** | Tag-based scoring yang mencocokkan preferensimu ke destinasi paling relevan |
| рҹҰ… **8 Tipe Kepribadian** | Temukan tipe travelermu: Explorer, Beach Lover, Culture Seeker, dan lainnya |
| рҹҢҷ **Dark & Light Mode** | Toggle tema yang tersimpan di localStorage вҖ” nyaman di mata kapanpun |
| рҹҺЁ **Animasi Premium** | Gradient orbs, progress bar glow, slide transitions, floating particles |
| рҹ“ұ **Fully Responsive** | Tampilan optimal di semua ukuran layar вҖ” mobile, tablet, desktop |
| рҹ”— **Google Maps Integration** | Klik destinasi вҶ’ langsung buka lokasi di Google Maps |
| рҹ“Ө **Native Share** | Share hasil quiz via Web Share API atau copy to clipboard |
| вҡЎ **Zero Dependencies** | Murni HTML + CSS + JS. Tidak perlu npm install apapun |

## вҡЎ Quick Start

### Cara 1 - Buka Langsung (Paling Mudah)

```bash
# Clone repo
git clone https://github.com/agusadhitama/wanderlust-indonesia.git

# Masuk folder
cd wanderlust-indonesia

# Buka di browser (double-click index.html, atau:)
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux
```

### Cara 2 - Pakai Live Server (Recommended untuk Development)

```bash
# Install Live Server (jika belum)
npm install -g live-server

# Jalankan
live-server .
```

> Atau pakai ekstensi **Live Server** di VS Code вҶ’ klik kanan `index.html` вҶ’ *Open with Live Server*

## рҹӨқ Kontribusi

Kontribusi sangat welcome! Ada beberapa cara kamu bisa bantu:

### рҹ—әпёҸ Tambah Destinasi Baru

Edit `js/data.js`, tambahkan objek baru ke array `DESTINATIONS`:

```javascript
{
  id: "nama-unik",
  name: "Nama Destinasi",
  province: "Provinsi",
  emoji: "рҹҸқпёҸ",
  gradient: "linear-gradient(135deg, #warna1, #warna2)",
  desc: "Deskripsi singkat dan menarik tentang destinasi ini.",
  tags: ["tag1", "tag2", "tag3"],         // untuk kategorisasi
  score_tags: ["tag1", "tag2", "hiking"], // untuk scoring algorithm
  vibes: ["рҹҺҜ Aktivitas 1", "рҹҢ… Aktivitas 2", "рҹҸ•пёҸ Aktivitas 3"]
}
```

### вқ“ Tambah Pertanyaan

Edit array `QUESTIONS` di `js/data.js`:

```javascript
{
  id: 21,
  category: "Kategori Baru",
  text: "Pertanyaan yang menarik?",
  options: [
    { emoji: "рҹҺҜ", label: "Opsi A", desc: "Deskripsi singkat", value: "nilai", tags: ["tag"] },
    // ... 3 opsi lainnya
  ]
}
```
